CartLens performs market-basket analysis on retail transaction data to surface association rules (e.g., "people who bought X also bought Y"), rank the top rules by lift, and present actionable business insights in an interactive Streamlit dashboard.

This project is useful for merchandising, cross-sell recommendations, bundle suggestions, and exploratory analysis of transaction datasets.

## Key features
- Ingests market transaction data 
- Generates frequent itemsets and association rules 
- Ranks top rules by support, confidence, and lift
- Business insights panel (top rules, recommended bundles, product pairs, segment suggestions)
